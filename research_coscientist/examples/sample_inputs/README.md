# Research Co-Scientist Sample Inputs

This synthetic packet demonstrates evidence-based engineering research. It contains no operational controls, facility data, personal data, or instructions to change a live system.

The sample asks the co-scientist to identify testable, non-clinical ways to
reduce energy use in a small data-center cooling loop. It includes a small
synthetic baseline table so hypotheses can name measurable variables and
confounders, but it intentionally omits intervention outcomes and manufacturer
limits. The workflow must preserve those gaps, counterarguments, and the
human-review boundary.
