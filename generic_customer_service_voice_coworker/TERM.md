# Terms and Operating Notes

This blueprint is for local customer-service assistance and testing. It is not a
replacement for trained support staff, emergency dispatch, legal counsel,
medical professionals, account-security operations, or billing approval flows.

The customer is responsible for the knowledge text it provides and for reviewing
conversation artifacts before using the co-worker with real customers.

The blueprint may start or reuse the shared NVIDIA services on Spark. It does
not modify Spark code directly and does not stop the shared NVIDIA stack during
post-launch cleanup.
