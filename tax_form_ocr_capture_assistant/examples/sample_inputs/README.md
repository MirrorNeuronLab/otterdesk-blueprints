# Sample Inputs

This folder includes a lightweight demo subset extracted from a public Hugging Face shard derived from NIST-SD2 tax forms.

- Demo source: https://huggingface.co/datasets/hyturing/US_tax_forms_donut
- Official reference source: https://www.nist.gov/srd/nist-special-database-2
- Alternate official reference source: https://www.nist.gov/srd/nist-special-database-6

Bundled files:

- `us_tax_form_sample_0001_sch_e_2.png`
- `us_tax_form_sample_0001_sch_e_2.json`
- `us_tax_form_sample_0002_sch_e_2.png`
- `us_tax_form_sample_0002_sch_e_2.json`
- `SAMPLE_DATASET_MANIFEST.json`
